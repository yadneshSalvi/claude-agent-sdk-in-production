# Claude Agent SDK in Production

Companion code for the series **[Claude Agent SDK in Production: Build an AI Data Analyst with FastAPI and Next.js](https://yadneshsalvi.com/series/claude-agent-sdk-fastapi-nextjs)** — an AI data analyst that reads your CSVs, writes and runs its own Python, and hands back charts and reports, built on the engine inside Claude Code.

Every folder below is the **complete, runnable project as it exists at the end of that part**. You can `cd` into any part and run it without assembling anything from earlier parts. All code was tested end to end against the live Claude Agent SDK before publishing. Each part folder also contains a short `demo.mp4` screen recording showing exactly what that part looks like when it runs.

## The sample data: Beanline

The series analyzes **Beanline**, a fictional specialty-coffee chain: six stores, twelve products, six months of daily sales (11,000+ rows). The dataset is generated by a deterministic script — same bytes on every run, on every machine — so the numbers in the series are reproducible on yours:

```bash
cd data
python3 generate_beanline.py
```

Each part folder ships with the generated CSVs already in place; you never need to run the generator unless you want to.

## Running a part

Each part's own `README.md` has the exact steps. The backend uses [uv](https://docs.astral.sh/uv/):

```bash
cd part-0N-*/backend
uv sync
uv run python agent.py        # Part 1; later parts: uv run uvicorn app.main:app
```

**Frontend** (Parts 3+):

```bash
cd part-0N-*/frontend
npm install
cp .env.local.example .env.local
npm run dev
```

## Authentication

The Agent SDK uses your existing **Claude subscription login** (run `claude` once and log in) or an `ANTHROPIC_API_KEY`. [Part 1](https://yadneshsalvi.com/blog/agent-sdk-1-first-agent) covers both paths and how to cap spending. Keys, if you use one, go in `backend/.env` (gitignored). Never commit them.

## Versions

Pinned in each part's `pyproject.toml` / `uv.lock` — the exact versions the series was written and tested against: Python 3.13, `claude-agent-sdk` 0.2.110, FastAPI 0.136.x, Next.js 16.x, Node 22.
